# GrowMe
It's an interactive self develpment that combines learning soft skills , a test , camera challenges and a certificate
Video on how to use it (https://drive.google.com/file/d/1lGcpwC5dZL5al-KXN42Y8KruaJMHemyj/view?usp=sharing)

Live demo (https://growme-delta.vercel.app/)

I didn't use AI tools excepts for basics code and I fixed all errors and added to it

Features :
   Sign up and log in using lacal storage
   A test 10 questions
   Youtube lessons for each skill
   Camera challenges
   Generated certificate
   PWA

Built with :
   Frontend : HTML , CSS , JS
   Local storage 
   WebRTC(camera access)
   PWA Manifest.json , Server worker (sw.js)

Built for the Horizons Hack Club (to Canada)
